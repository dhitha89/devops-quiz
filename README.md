# DevOps Quiz Hub

Interactive quizzes companion to [The DevOps Playbook](https://devops-playbook.hashnode.dev) by [@sharmilasait](https://hashnode.com/@sharmilasait).

Each blog post gets a quiz. Pick a topic, answer questions, and get instant feedback with explanations.

Built with Laravel 12, Blade, Tailwind CSS and vanilla JS. Dark/light mode supported.
